# Paste-in inbox — the Phase-1 workhorse

Drop ANY job listing here (Coroflot, Internshala, LinkedIn, a studio page, a
WhatsApp forward — anywhere). On the next run it flows through the same
score → packet → digest pipeline. Nothing here is applied automatically.

Format — one block per job, separated by a line with just three dashes:

    ### Company Name — Role Title
    url: https://link-to-apply
    location: Mumbai
    Paste the job description text here.

Only text after the marker line below is read. Add your jobs under it.
